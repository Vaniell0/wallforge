# Wallforge documentation

Project docs, organised by audience.

## For users

- [**Configuration reference**](configuration.md) — every field in
  `config.json` and every Home Manager option, with defaults and
  examples.
- [**Troubleshooting**](troubleshooting.md) — common failure modes
  (Steam not detected, preview not rendering, workspace daemon not
  firing) and their fixes.
- [**Changelog**](changelog.md) — what shipped in each release.

## For contributors

- [**Architecture**](architecture.md) — package graph, data flow,
  the stdlib-first philosophy.
- [**Backends**](backends.md) — deep dive on swww, mpvpaper, and
  linux-wallpaperengine: how wallforge drives each, quirks to know
  about, how to add a new one.
- [**Contributing guide**](contributing.md) — dev setup, testing
  patterns, commit conventions, release process.
